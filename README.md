# Jobs @ Covenant

Covenant is an autonomous surveillance station powered by edge AI, solar energy, and drone autonomy. This repo contains technical challenges across the disciplines that make it real.

Pick the challenge that matches your background, build something, and send your submission to [eduardo@nuclea.solutions](mailto:eduardo@nuclea.solutions).
